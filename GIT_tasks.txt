При выполнении задания не стесняйтесь обращаться к документации по git или хотя бы гуглу. Без этого никак ;)
Лучше сначала полностью прочитать задание, и понять, какие git команды вам понадобятся, а также изучить их.

0. Установите гит в вашу операционную систему. Работайте с гитом используя командную строку (например, bash, входящий в git for windows).
1. Создайте отдельную директорию с именем Git, в которой вы будете работать. Там необходимо инициализировать новый пустой репозиторий.
2. Создайте текстовый файл с любым содержанием. Лучше чтобы он имел несколько строк. Сделайте коммит (сохраните состояние).
Помните, что сообщение к каждому коммиту долно ясно давать понять, какие изменения были внесены. 
3. Создайте новую ветку с любым именем.
4. Добавьте еще один файл, сделайте коммит.
5. В этой же ветке внесите изменения в первый файл в какую-нибудь строку, сделайте коммит. Убедитесь, что статус репозитория чист.
6. Переключитесь обратно в ветку master, измените первый файл в другом месте. Сделайте коммит.
